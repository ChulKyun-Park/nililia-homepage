# 레이아웃 기준 가이드

## 기준점: 홈 히어로 섹션

모든 섹션의 콘텐츠 좌우 끝단은 홈 히어로 섹션을 기준으로 정렬한다.

### 좌측 끝단
- 히어로 텍스트 박스의 좌측 시작점
- 코드 기준: `max-w-7xl` 컨테이너 + `px-6` + `lg:pl-12`

### 우측 끝단
- 히어로 카드 애니메이션 박스의 우측 끝점
- 코드 기준: `max-w-7xl` 컨테이너 + `px-6` 기준 우측 끝 (애니메이션이 `right-0`)

### 적용 규칙
- 섹션에 들어가는 컨텐츠가 몇 단이든 (2단, 3단, 5단 등):
  - 첫 번째 단의 좌측 끝 = 히어로 텍스트 좌측 끝단과 일치
  - 마지막 단의 우측 끝 = 히어로 애니메이션 우측 끝단과 일치
- 섹션 자체의 배경(bg)은 화면 전체 너비를 사용해도 됨
- 안쪽 콘텐츠만 이 기준에 맞춘다

### 참조 코드 (Hero.tsx)
```
// 컨테이너
<div className="relative z-10 mx-auto max-w-7xl px-6 py-4 lg:py-6">

  // 텍스트 박스 (좌측 기준)
  <div className="max-w-xl lg:pl-12">

  // 애니메이션 박스 (우측 기준)
  <div className="absolute right-0 top-1/2 -translate-y-1/2">
```

### 실무 적용
콘텐츠 그리드를 감싸는 래퍼에 아래 클래스를 적용:
```
className="mx-auto max-w-7xl px-6 lg:pl-12"
```
이렇게 하면 좌측은 히어로 텍스트와 정렬, 우측은 컨테이너 끝과 정렬된다.
